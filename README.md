# ASP.Net CRUD Application

<small>
A simple ASP.NET Core MVC CRUD (Create, Read, Update, Delete) application for managing student records.  
This project is built using Entity Framework Core (Code First Approach) and demonstrates basic database operations.
</small>

## 🚀 Features
- Add new students
- Edit existing student details
- Delete student records
- View student list with details
- Validation for inputs (Name, Email, Phone, DOB)

---

## 🛠️ Technologies Used
- ASP.NET Core MVC
- Entity Framework Core (Code First)
- SQL Server
- Bootstrap (for UI styling)

---

## 📂 Project Structure
- **Models/** → Student model (ID, Name, Dob, Email, Phone)
- **Controllers/** → StudentController with CRUD actions
- **Views/** → Razor views for List, Create, Edit, Delete, Details
- **Migrations/** → EF Core migrations for database schema
